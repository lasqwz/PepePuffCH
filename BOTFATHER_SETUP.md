# Настройка полноэкранного режима в @BotFather

Отправь эти команды @BotFather:

1. `/mybots` - выбери своего бота @PepePuffBot

2. `Bot Settings` → `Menu Button` → `Edit Menu Button`
   - URL: `https://pepepuffch-production.up.railway.app`
   - Text: `Открыть магазин` (или оставь пустым)

3. `Bot Settings` → `Inline Mode` → `Turn on`
   - Это позволит боту работать в полноэкранном режиме

4. Дополнительно можешь настроить:
   - `Edit Description` - описание бота
   - `Edit About` - краткое описание
   - `Edit Botpic` - аватарка бота

После этих настроек приложение будет открываться на весь экран без шапки Telegram!
