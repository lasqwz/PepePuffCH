# 🚀 Быстрый старт PepePuff Bot (локально)

## Шаг 1: Установите ngrok

Если ngrok еще не установлен, следуйте инструкциям в `INSTALL_NGROK.md`

Или быстро:
```bash
# Скачайте с https://ngrok.com/download
# Распакуйте и переместите в /usr/local/bin/
```

## Шаг 2: Запустите бота

```bash
./start.sh
```

Вот и всё! 🎉

Скрипт автоматически:
- ✅ Проверит все зависимости
- ✅ Запустит ngrok
- ✅ Обновит URL в .env
- ✅ Запустит бота

## Что дальше?

1. Откройте Telegram
2. Найдите вашего бота
3. Отправьте `/start`
4. Нажмите "Открыть магазин"
5. Готово! 🛒

## Полезные ссылки

- **ngrok Dashboard**: http://localhost:4040 - смотрите все запросы
- **Логи бота**: `logs/bot.log`
- **Остановка**: `Ctrl+C`

## Проблемы?

Смотрите `START_LOCAL.md` для подробной информации
